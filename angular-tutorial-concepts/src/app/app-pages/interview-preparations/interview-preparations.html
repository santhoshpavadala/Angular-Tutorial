CHANGE DETECTION & PERFORMANCE

9. How does Angular change detection work?

Angular runs a top-down check of components to detect data changes. Zone.js triggers detection when async tasks complete.

10. What is OnPush strategy?

OnPush runs change detection only when: - Input reference changes - Event occurs - Observable emits - markForCheck() is called

11. Why is OnPush better for large apps?

It reduces unnecessary DOM checks and improves performance.

12. What are common performance mistakes?

  • Not using trackBy
  • Mutating objects instead of replacing
  • Too many subscriptions
  • Heavy logic in templates
