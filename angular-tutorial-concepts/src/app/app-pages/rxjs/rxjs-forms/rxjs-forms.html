Rxjs In Reactive form: ValueChanges | combineLatest | statusChanges

Value Changes

valueChanges is an Observable that emits whenever the value of a form control or form group changes. It emits the latest value.

Status Change

statusChanges emits whenever the validation status of a form changes.

combineLatest

combineLatest is an RxJS operator used to combine multiple observables and emit the latest values from all of them whenever any one changes.
Since you are working with Angular Reactive Forms, it is very commonly used to listen to multiple form controls together.
