Rxjs-Operators

Opearors:

RxJS operators are functions that transform, filter, combine, or control Observables.

Types of Opearors:

  • 1. Creation Operators
  • 2. Pipeable / Transformation Operators
  • 3. Filtering Operators
  • 4. Combination Operators
  • 5. Error Handling Operators
  • 6. Utility / Multicasting Operators

Creation Operators:

  • of()
  • from()
  • interval()
  • timer()
  • range()
  • fromEvent()

Transformation Operators:

  • map()
  • mergeMap() (flatMap)
  • switchMap()
  • concatMap()
  • exhaustMap()

Filtering Operators:

  • filter()
  • take()
  • takeUntil()
  • debounceTime()
  • distinctUntilChanged()

debounceTime Operator

debounceTime() waits for a pause in emissions and then emits only the latest value.

Syntax: debounceTime(timeInMs)

Merge Operator

merge() combines multiple Observables into one Observable and emits values as soon as any source emits.

Syntax: merge(obs1$, obs2$, obs3$)

{{ item }}

Take Operator

take() limits the number of values emitted by an Observable and then automatically completes it.

Syntax: take(count)

{{item}}

Pluck Operator

pluck() extracts a specific property (or nested property) from each emitted object.

“Pick this field from every emitted object”

Syntax: pluck('propertyName')
