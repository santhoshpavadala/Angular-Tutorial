Angular - Data Binding Examples

What is Data Binding?

Data Binding is the process of connecting component data with the template to keep the UI synchronized automatically.

    Data binding is two types in angular:
  • One-Way Data Binding in Angular
  • Two-Way Data Binding

One-Way Data Binding

One-Way Data Binding means data flows in only one direction between the component and the template.

component to template

templalate to component

But not both simultaneously.

Types of One-Way Data Binding

  • Interpolation
  • Property Binding
  • Event Binding

1.1 Interpolation

Interpolation is a one-way binding technique used to display data from the component to the template.
It uses double curly braces.

Interpolation is a one-way data binding technique used to display component data in the template using double curly braces syntax.

Event Binding

Event Binding allows the template to send user actions to the component.

    It listens to events such as:
  • click
  • input
  • change
  • mouseover
  • keyup

Examples

Two-way data binding allows data to flow both directions between component and template.
