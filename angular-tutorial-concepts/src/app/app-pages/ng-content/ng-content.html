Angular - Ng-Content

ng-content

ng-content is used to project content from a parent component into a child component template.
It allows a component to accept external HTML content.

ng-container

ng-container is a logical container that does not create a DOM element.

ng-container tag will not appear in console and it is imaginary div and uses for conditions instread of writing in tr.

Why ng-container is Needed

Angular does not allow two structural directives on the same element.

for example: div *ngIf="user" *ngFor="let item of list"

so thats y in table we can use ng-Container

ng-template

ng-template defines a template that Angular does not render immediately.

It is rendered only when Angular explicitly tells it to render.

by default it wont display anything in ui.
