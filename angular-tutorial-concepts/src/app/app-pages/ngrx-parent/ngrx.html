Angular NGRX

What is NgRx?

NgRx is a state management library for Angular applications based on the Redux pattern and powered by RxJS Observables.

Why Do We Need NgRx?

In small apps:

  • We use Input(), Output()
  • Services with BehaviorSubject
  • Direct data sharing

In large apps:

  • Multiple components need same data
  • Complex API flows
  • State must persist across routes
  • Debugging becomes hard

NgRx solves:

  • Centralized state
  • One-way data flow
  • Predictable state updates
  • Easy debugging with DevTools

Data Flow (One-Way)

  • Component dispatches Action
  • Action goes to Reducer
  • Reducer updates Store (state)
  • Component selects data from Store
  • If API needed â†’ handled by Effects

Core Concepts of NgRx:

1. Store

  • Central state container
  • Immutable
  • Read-only (can only change via actions)

2. Actions

Actions describe what happened

3. Reducer

Reducer defines how state changes

4. Selectors

Used to read data from store

5. Effects (For API Calls)

Effects handle:

  • HTTP requests
  • Side effects
  • Async operations

Installing NgRx

ng add ngrx/store
ng add ngrx/effects
ng add ngrx/store-devtools
