In Angular, a Service is a class used to share data, logic, or functionality across multiple components.

It follows the Singleton pattern (when provided in root), meaning one shared instance across the application.

    We use services for:
  • API calls (HTTP requests)
  • Business logic
  • State management
  • Data sharing between components
  • Reusable utility functions
  • Authentication & authorization logic
