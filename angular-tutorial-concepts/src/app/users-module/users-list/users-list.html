Angular - Modules

What is an Angular Module (NgModule)?

An Angular Module is a logical container that groups related components, directives, pipes, and services to organize an application and enable features like lazy loading and separation of concerns.

    Key Points
  • Introduced in early Angular versions
  • Uses NgModule decorator
  • Optional in Angular 15+ (standalone-first)
  • Still widely used in enterprise & legacy apps
  • Mostly used today for lazy-loaded feature modules

What is Lazy Loading?

Lazy loading is a technique where Angular loads feature code only when the user navigates to a specific route, instead of loading everything at application startup.
